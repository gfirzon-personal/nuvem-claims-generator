```sh
pip install pipenv
```

```sh
pipenv --version
```

Install requirements using req
```sh
pipenv install -r requirements.txt
```

```sh
pipenv shell
```

Displays a dependency graph of all installed packages in the Pipenv environment
```sh
pipenv graph
```

Delete the existing virtual environment:
```sh
pipenv --rm
```
